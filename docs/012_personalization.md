# 012 パーソナライズ・レコメンド

## 概要
スワイプ履歴・お気に入りからユーザーの嗜好スコアを計算し、「あなたへのおすすめ」フィードを生成する。協調フィルタ＋コンテンツベースのハイブリッド推薦。

## 依存
- 001 Supabase 初期設定
- 007 お気に入り
- 009 スワイプフィード
