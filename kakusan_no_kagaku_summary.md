# 拡散の科学：Anatomy of Retweet まとめ

アップロードされた資料『拡散の科学（Anatomy of Retweet）』に基づき、Twitter（現X）における拡散のメカニズム、ユーザーがリツイートを行う心理的背景、およびそれを加速させる要素について整理したマークダウンレポートです。

---

## 1. OVERVIEW：そもそも拡散とは

### 1-1. Twitterの現状と「世の中ごと」への昇華
* **リアルタイムの会話の場**: Twitterは、世界中で起きている出来事とそれにまつわる会話がリアルタイムで繰り広げられる場所です。
* **市場規模（2021年Q4時点）**: Monetizable Daily Active Usage (mDAU) は**2億1千万人以上**。日本は売上第2位の非常に重要なマーケットです。
* **世の中ごとへの昇華**: 生活者のリアルタイムな話題や熱量が集まることで、Twitter上の単なる話題が社会現象（世の中ごと）へと昇華します。
  * *具体事例*：リュウジ氏（@ore825）が投稿した「じゃがアリゴ」のレシピツイート（15.2万RT、48.2万いいね）は、あまりの反響にカルビー公式が「じゃがりこ じゃがバター」を一時休売にするほど、実際の購買や生産ラインにまで影響を及ぼしました。

### 1-2. 拡散の鍵を握る「熱量伝播」
* **熱量伝播の定義**: リツイートによって、人の気持ちを動かす1つのツイートが、コミュニティ（フォロワーの枠）を越えて拡がっていく現象を指します。
* **拡散の波及力**: 1リツイートあたりの波及力（インプレッション）は、およそ **300 imps** （平均329 imps、中央値323 imps）に相当します。
* **拡散の目安と「0.1%の事象」**:
  * 日本国内で拡散された（1回以上リツイートされた）ツイートの **99.9% は1〜1,299リツイートの間** に収まります。
  * 投稿主自身が肌感でバズを察知し「バズったので…」と返信（宣伝など）をぶら下げ始めるリツイート数の中央値が **1,300リツイート** です。
  * すなわち、**1,300リツイート以上を達成することは「0.1%の限られた事象」** であり、これこそが「世の中ごと」へと繋がる拡散の入り口（目安）となります。
  * 本研究は、経験や天才といった属人性・偶然に頼るのではなく、この「0.1%の可能性」に科学的にチャレンジするための仕組みを解明したものです。

## 2. WHY：なぜ人は拡散するのか（感情の分析）

### 2-1. 分析方法とデータソース
* **対象期間**: 2019年1月〜2020年8月
* **分析データ**: 
  * 1,300RT以上を達成したツイート（バズった投稿）：5,100件
  * 1,300RT未満のツイート（拡散しなかった投稿）：5,100件（ランダム抽出）
* **解析アプローチ**: 
  * 自然言語処理手法（トピックモデル：Latent Dirichlet Allocation）を用い、バズったツイートに寄せられた返信や引用ツイートのテキストから「感情のかたまり（トピック）」を抽出。
  * 機械（定量データ・回帰分析）と人間（定性的な解釈・意味付け）の力を掛け合わせることで、リツイートボタンを押したくなるユーザーの心理を **「16の熱量」** に分類。それらがさらに **「6つの熱量伝播パターン」** へとまとまることを突き止めました。

## 3. 「6つの熱量伝播」と「16の熱量」の詳細

ユーザーがリツイートボタンを押したくなるモチベーション（熱量）は、以下の6つの軸・16の要素に分解されます。※インセンティブ以外の各熱量は、バズった際はいずれも**平均6万リツイート以上**を記録しており、感情の種類による優劣はありません。

### ① 直感で拡がる
> **ワンフレーズ：「反射的にリツイート！自分が感じたものを共有したい」**

* **WOW（驚き）** [熱量シェア: 23% / 平均RT数: 65K]
  * *特徴*: 既視感のないレアな出来事や、発見・遭遇・目撃の喜び。「信じられない」「やばい」「神」といった、笑いより驚愕が勝る内容。画像・動画の割合が高い（画像48%、動画40%）。波形は一過性の「パルス型」。
* **FUN（面白い・楽しい）** [熱量シェア: 23% / 平均RT数: 66K]
  * *特徴*: ワクワク感、ネタ感、思わず一言ツッコミたくなる投稿。画像とテキストのギャップなどが有効。オーディエンスは「wwww」「草生える」と反応。フォーマットは画像51%、動画38%。波形は「パルス型」。
* **尊い（リスペクト・好きすぎる）** [熱量シェア: 11% / 平均RT数: 67K]
  * *特徴*: 単なる驚きを超え、神聖さやリスペクトを感じる投稿。コアファンにしか分からないこだわりや圧倒的なクリエイティブ力。画像52%、動画37%。波形は「パルス型」。
* **癒し（可愛さ・ほっこり）** [熱量シェア: 13% / 平均RT数: 67K]
  * *特徴*: 動物、赤ちゃん、心が和むほっこりする良い話。動画のパワーが非常に強く、動画割合が画像と並ぶ（画像46%、動画46%）。波形は「パルス型」。
* **感動（ジーンとくる・洗われる）** [熱量シェア: 6% / 平均RT数: 67K]
  * *特徴*: 実話エピソード、感謝、心が温まったり洗われたりする内容。「泣ける」「素晴らしい」「ありがとう」といった反応。画像投稿が多め（59%）。波形は「パルス型」。
* **ショック（悲しみ・やるせなさ）** [熱量シェア: 3% / 平均RT数: 82K]
  * *特徴*: 多くの人が心を痛める内容、現実を受け止めきれないつらいニュース。「嘘でしょ」「頭が追いつかない」という反応。文字のみの投稿割合も高い（41%）。波形は急激に立ち上がる「スパイク型」。

### ② 知識で拡がる
> **ワンフレーズ：「これ知っておいて！情報をみんなに知ってほしい」**

* **知っトク（ライフハック・有用性）** [熱量シェア: 11% / 平均RT数: 62K]
  * *特徴*: 日常生活で使えそう、気軽に試せそうなライフハックや新しい気づき（膝ぽん感）。「いいこと聞いた」「今度やってみよう」と思わせる内容。画像54%、テキスト24%。波形は「パルス型」。
* **注意喚起（リスク回避）** [熱量シェア: 7% / 平均RT数: 65K]
  * *特徴*: 詐欺、違法、損、危険など、知ることでリスクを回避できる身近な情報。「みんな気をつけて」「騙されないで」と拡散される。テキストのみが37%と多め。波形は「パルス型」。

### ③ 主張で拡がる
> **ワンフレーズ：「自分も同じこと考えてる！他人の意見を借りて自分の意見を主張したい」**

* **同調（代弁・擬似的な吐露）** [熱量シェア: 6% / 平均RT数: 62K]
  * *特徴*: 誰かが自分の気持ちを代弁してくれている感覚。リツイートすることで第三者の言葉を自分の意見として提示（擬似的な吐露）する。「ほんとそれ！」「実は私も」。テキストのみが43%と高比率。波形は「パルス型」。
* **物申す（世論へのアンチテーゼ）** [熱量シェア: 11% / 平均RT数: 62K]
  * *特徴*: 世の中や世論がおかしい方向へ流れていると感じた際、その流れを断ち切るような強い主張。「よくぞ言ってくれた」「これはおかしい」と拡散。テキストのみが43%。波形は「パルス型」。

### ④ 納得で拡がる
> **ワンフレーズ：「めっちゃわかる！自分と同じ価値観に触れられて嬉しい」**

* **あるある（日常の共感）** [熱量シェア: 8% / 平均RT数: 66K]
  * *特徴*: 多くの生活者が過去に同じ経験をしたことがあるような同意・共感。「めっちゃわかる」「それな」「私だけじゃなかった」。画像46%、動画32%、テキスト22%。波形は「パルス型」。
* **真理（言語化の妙・名言）** [熱量シェア: 7% / 平均RT数: 62K]
  * *特徴*: 潜在的に思っていたけれど、誰も言語化できなかったものをズバッと表現した投稿。「言い得て妙」「名言」と大きく頷かせる。テキストのみが53%と過半数を占める。波形は「パルス型」。

### ⑤ 声援で拡がる
> **ワンフレーズ：「パワーよ届け！誰かの・何かの力になりたい」**

* **応援（ファン活動）** [熱量シェア: 14% / 平均RT数: 69K]
  * *特徴*: 好きな人やコンテンツ、推しの応援・力になりたいという高い熱量。「頑張れ！」「ずっとついていきます」。画像55%、テキスト30%。波形は「スパイク型」。
* **支援（助け合い・同情）** [熱量シェア: 4% / 平均RT数: 78K]
  * *特徴*: 同情してしまうリアルな未解決トラブルなど、リツイートが誰かの助けにつながる投稿。「ひどい」「みんなで助けたい」。テキストのみが51%と非常に多い。波形は「パルス型」。

### ⑥ 欲求で拡がる
> **ワンフレーズ：「欲しい！やりたい！自分の欲望を満たしたい」**

* **WANT（潜在的物欲・興味）** [熱量シェア: 2% / 平均RT数: 66K]
  * *特徴*: リツイートで手に入るわけではないが、純粋に「欲しい」「やってみたい」「買いたい」と思わせるモノ・コト。画像のインパクトが極めて重要（画像61%）。波形は「パルス型」。
* **インセンティブ（キャンペーン）** [熱量シェア: 28% / 平均RT数: 112K]
  * *特徴*: フォロー＆RTキャンペーンなど、物質的なメリットがある投稿。シェア率・平均RT数（11.2万RT）ともに全熱量の中で最大。波形は「スパイク型」。

## 4. データから見える重要な知見

### 4-1. 熱量の種類に優劣はない
キャンペーンである「インセンティブ」は別格（平均11.2万RT）ですが、それ以外の純粋な感情系熱量（WOW、FUN、応援、癒し、知っトクなど）は、シェアの大きさ（2%〜23%）に関わらず、**すべてバズれば平均6万リツイート以上を等しく記録**しています。
特定の感情が拡散に有利ということはなく、コンテンツやブランドの性質に合った適切な「熱量」を刺激することが重要です。

### 4-2. クリエイティブ表現（フォーマット）の適切な使い分け
全体傾向としては画像付き（Image）が主流ですが、刺激したい熱量によって最適な表現方法（フォーマット）が明確に異なります。
* **動画（Video）が効く熱量**: 『癒し』（動画46%）、『WOW』（動画40%）、『FUN』（動画38%）、『尊い』（動画37%）などの直感に訴えるもの。動物やクリエイティブの凄さを伝えるには動画が必須。
* **テキストのみ（Text-only）が効く熱量**: 『真理』（53%）、『支援』（51%）、『物申す』（43%）、『同調』（43%）など。言葉そのものの重みや、代弁性の高さ、情報の深刻さを伝えるには、装飾のないテキストが最もオーディエンスに響きます。
